# Images Assets

This folder contains all image assets for the XSAV Lab website.

## Folder Structure

- **`team/`** - Team member profile images for About section
- **`logo/`** - Company logos and branding (future)
- **`projects/`** - Project screenshots and case study images (future)
- **`backgrounds/`** - Background images and textures (future)

## Image Optimization

Please optimize all images before adding them:
- Use appropriate formats (WebP for photos, SVG for icons)
- Compress images to reduce file size
- Use responsive images when possible

## Tools for Optimization

- [TinyPNG](https://tinypng.com/) - PNG/JPG compression
- [Squoosh](https://squoosh.app/) - Image optimization
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - SVG optimization
