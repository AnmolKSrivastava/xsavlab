# XSAV Lab - AI-Powered Enterprise Solutions

A stunning, production-ready landing website for XSAV Lab, showcasing cybersecurity services, cloud solutions, and AI agent development.

## 🚀 Features

- **Next-Generation UI**: Dark, futuristic design with glassmorphism effects
- **Smooth Animations**: Powered by Framer Motion for butter-smooth interactions
- **Interactive AI Demo**: Live chatbot interface with simulated responses
- **AI Agent Marketplace**: Browse and explore pre-built AI solutions
- **Fully Responsive**: Mobile-first design with sticky bottom action bar
- **Modern Tech Stack**: React, Tailwind CSS, Framer Motion, Lucide Icons

## 📦 Tech Stack

- **React 18** - UI framework
- **Tailwind CSS 3** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Beautiful icon set

## 🎨 Design System

### Colors
- **Background**: `#0B0F19` (Dark navy/black)
- **Primary**: `#00D4FF` (Neon Blue)
- **Secondary**: `#7B61FF` (Purple)
- **Text**: White / Light Gray

### Features
- Glassmorphism cards with backdrop blur
- Glowing borders and hover effects
- Smooth transitions and animations
- Grid background patterns
- Gradient overlays

## 🚀 Getting Started

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```
